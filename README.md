# RezenSurvivalModpack

A modpack designed to make survival gameplay more immersive and challenging.

## Features
- Realistic temperature system affecting survival
- Water mechanics rework: No more 3-block or 4-block infinite water sources
- Seasonal changes impacting the world and gameplay
- Classic combat mechanics with new tools and weapon types
- Quality of life improvements for single-player survival

## Minecraft Version
- **Currently supports**: 1.21.11

## Important Notes
- Designed for **single-player survival** only.
- Not optimized for multiplayer — if used on a server, ensure all players have the same modpack installed.
- **Minimum RAM**: 4GB
- **Recommended RAM**: 6GB or more
